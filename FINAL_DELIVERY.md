## 🚀 Deploy to Vercel (5 Minutes)

### Step 1: Push to GitHub
```bash
git init
git add .
git commit -m "Initial portfolio website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/adithya-graphics.git
git push -u origin main
```

### Step 2: Deploy on Vercel
1. Go to [vercel.com](https://vercel.com)
2. Click "New Project"
3. Import your GitHub repository
4. Click "Deploy" (no configuration needed!)

### Step 3: Add Custom Domain
1. In Vercel dashboard → Settings → Domains
2. Add your domain (e.g., adithyasgraphics.com)
3. Follow DNS setup instructions
4. Live in minutes! 🎉

## 💡 Tips for Success

1. **Edit `lib/constants.ts` first** - This one file controls all content
2. **Use CSS variables** - Change colors in `globals.css` for site-wide updates
3. **Start with development mode** - `npm run dev` for instant feedback
4. **Mobile test early** - Responsive design is critical
5. **Keep animations smooth** - Already optimized, avoid adding too many
6. **Use Next/Image** - For better image optimization
7. **Backup before changes** - Use git to track changes
8. **Test before deploy** - Build locally first: `npm run build`

## 🔧 Troubleshooting Quick Fixes

| Issue | Solution |
|-------|----------|
| Images not showing | Check paths are relative to `/public/` |
| Styles not applying | Clear `.next` folder: `rm -rf .next` |
| Build fails | Delete node_modules and reinstall: `npm install` |
| Port 3000 in use | Use different port: `npm run dev -- -p 3001` |
| TypeScript errors | Check `tsconfig.json` is not modified |
